Forms Examples

On this page we will be talking about the different types of forms that can be used on an html page.

Types of forms

  • Text Boxes
  • Dropdown Boxes
  • Radio Buttons
  • Check Boxes
  • Text Area
  • Submit Button

Text boxes

With text boxes you will start with open and close "form" tags, then inout tag with type="text". If you give the input a name of "password" it with star out the characters to keep it hidden.

If you add a "placeholder" attribute to the input tag, you can insert gery text to tell the user what they shold be inputing into the input box.



Dropdown Boxes

Downdown boxes are also know as option boxes. Start with a "form" tag followed by "select" open and close tags. the put in "option" tags with a value (which is sent when that particular item is selected)


Radio Button

With a radio button you have multiple items to select from but you can only choose one. Buttoms must have the same "name" attribute to be grouped together and have one selected. The value is what is submited to the target.

Checkboxes

Text Area

In a "textarea" you can set up a box for the user to enter text, such as a comment box or special instructions. It gives the user a box where they can type text in to be sent to the target. It is used to collect comments. In the "form" tags use "textarea" tags and set the name to describe what it is collecting and can set a sizze using "rows=" and "cols=". Leave a space between the open and close "textarea" tag to make sure the box is empty

I do not belive that the place holder trick works here. You can simply put text between the "textarea" tags to write something into the box to start with. ex "comments", where "comments" will appear in the text box, which the user can delete and then write their comment.



Submit Button

The Submit button is an input used in each form to allow the user to send the info to the target to be processed. Simply put an "input" tag with the type of "submit" and a value which will be displayed on the submit button.
