Ordered lists

With ordered lists, they can be counted with numbers, 1, 2, 3
Or Uppercase letters, A, B, C
Lowercase letters, a, b, c
uppercase roman numerals I, II, III
Lower case roman numerals i, ii, iii
The Default is with numbers.
use "type="a" (A, a, i, I, 1) ect to set the type of ordered list you would like.

For Description Lists, you start with "dl" open and close. then add "dt" for data term followed by "dd" data description. the - has been added to the "dd" tag.
