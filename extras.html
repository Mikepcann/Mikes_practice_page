Extras


On this page I will show how you can add images to you HTML pages. The important feature here is the "img" tag that can be used to add images. This tag does not need a closing tag. It is followed by the "src" attribute which will link to an image saved in the "image" folder. The default when adding in multiple pictures to a page is to display them horizontally across the page. You could add in a "br" to break up the images and show them vertically.

Using the "atl" attribute within an "img" tag can add readable text if the image does not load to the page. Good feature for screen reader functions.
