Example Of Footer

To format tables to be clear and easy to manipulate with JS or CSS we can use "thead" to create the head of the table. Then use "th" to centre and bold those cell items. "tbody" will create the body of the the table. Start by using "tr" open and close and then inset the "td" to fill out the rows with information. You can then end the table with the "tfoot" to finish off the table.

Don't forget to add instructions to the "style" at the top of the page or in a seperate style.css to make the table have a border.
That can look like this: "style" table, td "{ border: 1px solid black}" and close the "style tag.
